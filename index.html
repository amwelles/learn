Assignments

So now that you have some skills, what should you actually do with them? Here are some very open-ended beginner projects that might help you get started down the path to theme development.

Create a Tumblr theme

Tumblr will introduce a few different concepts to your skillset. You'll learn about variables, some very basic programming logic (if/else statements, loops) and you should use your Tumblr to blog about whatever you're learning. It's great to keep a record so that you're able to go back and reference your own work. You can continue to update your theme and made small adjustments as time goes on.

  1. Sign up for a Tumblr account if you don't already have one.
  2. Visit your Tumblr URL (http://username.tumblr.com).
  3. Click "Edit theme" in the top, right-hand corner.
  4. Click "Edit HTML" in the top left sidebar.
  5. Click the gear in the top left (next to the back arrow).
  6. Select "Theme documentation" from the dropdown.
  7. Choose one:
    • Use the theme documentation to start from scratch.
    • Use Dash's tutorial to create a theme.
    • Use the Build Themes tutorial to create a theme.

Convert a Photoshop document into HTML & CSS

Unless you plan on designing and developing every site you create, you'll probably end up working with a Photoshop document that a designer has provided.

  1. Visit Freebiesbug and download a free PSD website template.
  2. Start a free 30-day trial of Adobe Photoshop.
